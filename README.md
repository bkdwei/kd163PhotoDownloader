# kd163PhotoDownloader
python3加pyqt5做的网易相册下载工具,适用于linux系统和window系统.

#使用方法
下载源代码
配置python3环境
使用pip安装pyqt5库
python3 kd163PhotoDownloader.py

# 依赖项
- pexpect
- pyqt5
- python3
